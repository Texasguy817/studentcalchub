# StudentCalcHub

Deploy-ready Next.js student tools website for studentcalchub.com.

## Deploy on Vercel
1. Upload this folder or zip to GitHub.
2. Import the GitHub repo into Vercel.
3. Connect `studentcalchub.com` in Vercel Domains.
4. In Namecheap, point DNS to Vercel as instructed.
5. After launch, submit `https://studentcalchub.com/sitemap.xml` in Google Search Console.
